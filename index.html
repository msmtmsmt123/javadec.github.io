JavaDec

Java decompiler

Welcome to JavaDec

JavaDec is a prototype of java decompiler. It supports only few basic language constructions.

Intellij IDEA plugin

You can try out IntelliJ Plugin that uses JavaDec. If you'd like to get the latest versions, add JavaDec plugin repository to IntelliJ plugin manager.

File > Settings > Plugins > Browse repositories... > Manage repositories...
Add this link http://javadec.github.io/updatePlugins.xml.

After installation classes will be decompiled automatically when you open them. If a class cannot be decompiled, a stub source will be generated by IntelliJ API Decompiler.

Online Java decompiler

If you don't use Intellij IDEA, try out online version of JavaDec.
